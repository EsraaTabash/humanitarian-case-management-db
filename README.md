# Humanitarian Case Management System

An end-to-end humanitarian case management system designed for NGOs and international organizations.

## Components

- SQL Server Database
- KoboToolbox Forms
- Python ETL Integration
- Power BI Dashboards

## Features

- Beneficiary Registration
- Household Assessments
- Case Tracking
- Service Referrals
- Follow-Up Visits
- KPI Monitoring

## Technology Stack

- SQL Server
- Python
- KoboToolbox
- Power BI
