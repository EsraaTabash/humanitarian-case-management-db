# Project Workflow

## 1. Data Collection

Field officers collect household and beneficiary data using KoboToolbox forms.

## 2. Data Storage

Collected data is stored in SQL Server relational tables.

## 3. Data Processing

SQL Views and Functions are used to prepare clean reporting datasets.

## 4. Access Control

Role-based permissions are applied for field officers, supervisors, managers, and Power BI users.

## 5. Reporting

Power BI dashboards use reporting views to monitor case status, referrals, vulnerability levels, and service delivery.
